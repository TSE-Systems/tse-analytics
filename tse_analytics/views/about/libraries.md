**Third-party libraries used by TSE Analytics**

TSE Analytics is built on the following free software libraries:

- [ConnectorX](https://github.com/sfu-db/connector-x/)
- [Great Tables](https://github.com/posit-dev/great-tables/)
- [LMFIT](https://lmfit.github.io/lmfit-py/)
- [Loguru](https://github.com/Delgan/loguru/)
- [Matplotlib](https://matplotlib.org/)
- [NodeGraphQt](https://github.com/jchanvfx/NodeGraphQt/)
- [NumPy](https://numpy.org/)
- [OpenPyXL](https://pypi.org/project/openpyxl/)
- [Pingouin](https://pingouin-stats.org/)
- [PtitPrince](https://github.com/pog87/PtitPrince/)
- [PyQt Toast](https://github.com/niklashenning/pyqttoast/)
- [PyQtGraph](https://www.pyqtgraph.org/)
- [PySide6-QtAds](https://pypi.org/project/PySide6-QtAds/)
- [PySide6](https://www.qt.io/)
- [SciPy](https://github.com/scipy/scipy/)
- [Traja](https://github.com/traja-team/traja/)
- [pandas](https://pandas.pydata.org/)
- [psutil](https://github.com/giampaolo/psutil/)
- [scikit-learn](https://github.com/scikit-learn/scikit-learn/)
- [seaborn](https://seaborn.pydata.org/)
- [umap-learn](https://github.com/lmcinnes/umap/)
- [xmltodict](https://github.com/martinblech/xmltodict/)
